# Getting Started

## Requirements

- Python 3.12+
- uv — `curl -LsSf https://astral.sh/uv/install.sh | sh`
- nmap, gobuster, nxc installed on your system

## Installation

### Use it

```bash
uv tool install git+https://github.com/StrafeNDestroy/strafekit.git
strafekit
```

### Develop it

```bash
git clone https://github.com/StrafeNDestroy/strafekit.git
cd strafekit
uv sync
source .venv/bin/activate
strafekit
```

## Navigation

```
Tab          move between panels
Arrow keys   navigate within a panel
Enter        select
q            quit
F1           help
```

# Commit Messages

Format: `type(scope): description`

| Type | When |
|---|---|
| `feat` | new feature |
| `fix` | bug fix |
| `docs` | documentation |
| `test` | tests |
| `refactor` | refactor |
| `chore` | deps, config |
| `ci` | CI changes |
| `scan` | scanning infrastructure |
| `creds` | credential management |

Examples:

```
feat(core): add Host domain model
fix(creds): handle duplicate entries
scan(nmap): add async wrapper
creds(sqlite): add domain filter query
docs(api): add Host docstrings
test(core): add Host unit tests
```
